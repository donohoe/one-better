one-better

Donate your blocked ads to charitable causes

View project onGitHub

Welcome to GitHub Pages.

It is estimated that 25% of ads are blocked by people installing browser extensions like AdBlocker. Wouldn't is be nice if we could channel this loss of brand advertising into a positive force for good?

This project allows you to setup alternative content that fits into old ad positions when ad-blocking is detected. It runs a random charitable ad with a single-call to action (donate, get involved, etc)

Example

<script src="src/app.js"></script>
<script type="text/javascript">

	var ads = new OneBetter();
	
	ads.set({
		"debug": true
	});

	ads.watch('#bigUnit div.ad', {
		unit: "square"
	});

	ads.watch('#whateverUnit .big', {
		unit: "wide"
	});

	ads.watch('#leader div', {
		unit: "leader"
	});
	
	ads.run();	
</script>

Demo

Square - These are standard square ads that can fill an entire space as necessary

Wide - These are wide ads, with a fixed height of 180px but 100% width

Leader - More in line with Leaderboard or Banner ads so best suited for ad positions that follow that

Customize

You can host your own version of this and change config variables as necessary. This may be required if AdBlocker rules are updated to block the version hosted here on Github.
